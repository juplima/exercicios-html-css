# Exercícios de HTML e CSS

Este repositório contém exercícios desenvolvidos durante aulas de desenvolvimento web,
com o objetivo de praticar conceitos básicos de HTML e CSS.

## Conteúdo
- Formulário de manutenção de usuários (HTML)
- Exercício de estilização com CSS
- Testes de inputs HTML e iframe

## Tecnologias utilizadas
- HTML
- CSS
